# Swim Set Repeat Timer
 
Swim Set Repeat Timer is a swim interval timer for swimmers and coaches who run repeatable swim sets. The app provides reliable iPhone-based timing with an Apple Watch companion display.

## Privacy Policy
 
Swim Set Repeat Timer does not collect, store, or share personal data.
 
- No accounts or login are required.
- No personal or device data is transmitted off the device.
- All timing data and workout logs remain on the user’s device unless explicitly shared by the user.
- The app does not use advertising, analytics, or third-party tracking.
